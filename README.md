# 🏏 IPL-Insights

Ball-by-ball IPL analysis (2008-2025) — player stats, 
team performance & match insights using Python & SQL

## 📊 Tools Used
- Python, Pandas, Matplotlib, SQL (SQLite)

## 🔍 Analysis Done
- Top 10 run scorers all time
- Top 10 wicket takers
- IPL winners each season
- Toss impact on match result
- Best venues by average score
- Virat Kohli season-wise runs

## 📁 Dataset
Ball-by-ball IPL data from Kaggle
